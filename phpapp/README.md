# Cookbook phpapp

Installs phpapp.

## Requirements

### Platform:

- Ubuntu/Debian
- RHEL and derivatives
- Fedora
- SLES / openSUSE
- ArchLinux

### Chef

- Chef 11+

### Cookbooks

- none

## Attributes

None.

## Usage

Put recipe[phpapp] in a run list, or include_recipe 'phpapp' to ensure that phpapp is installed on your systems.
